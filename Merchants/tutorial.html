Come accedere alla tua area privata

1. Scansiona il QR Code

Utilizza la fotocamera del tuo smartphone per scansionare il QR code che ti è stato fornito.

Questo ti aprirà direttamente la tua area personale.

2. Il QR non funziona?

Se il QR code non si apre correttamente, vai manualmente su questo link:

https://www.materacitypass.com/Merchants

Una volta aperta la pagina, inserisci il codice riportato sotto il QR code stampato nel foglio.

3. Cosa puoi fare una volta dentro?

Dalla tua area personale potrai:

  • Visualizzare la tua dashboard (numero transazione, grafici, ...)
  • Aprire lo scanner QR code per le convalide

3. Messaggi Scanner QR Code

I tipi di messaggi che possono comparire sono:

  • "Pass valido ✅" se si scansiona un pass attivo.
  • "QR Code non valido ❌" se si scansiona un QR code che non può più essere utilizzato e quindi richiede la rigenerazione.
  • "Pass non attivo ❌" se si scansiona un pass non attivo.
  • "Pass non trovato ❌" se si scansiona un QR code che non fa parte dei pass presenti nella piattaforma.

4. Messaggi Scanner QR Code per MUSEI

I tipi di messaggi che possono comparire sono:

  • "Pass valido ✅" con sfondo giallo se si scansiona un pass attivo gratuito oppure se il pass a pagamento ha terminato i ticket gratuiti.
  • "Pass valido ✅" con sfondo blu se si scansiona un pass attivo a pagamento con dei biglietti gratuiti disponibili all'interno.
  • "QR Code non valido ❌" se si scansiona un QR code che non può più essere utilizzato e quindi richiede la rigenerazione.
  • "Pass non attivo ❌" se si scansiona un pass non attivo.
  • "Pass non trovato ❌" se si scansiona un QR code che non fa parte dei pass presenti nella piattaforma.
